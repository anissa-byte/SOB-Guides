Sample Guide

A template to show the structure of a one-page guide.

Overview

This is a sample guide. Replace this content with your own. Each guide is a standalone HTML file stored in the guides/ folder.

Key Steps

  1. Create a new HTML file in the guides/ folder.
  2. Use this file as a template or write your own from scratch.
  3. Add an entry to guides.json in the root of the repo.
  4. Commit and push your changes.
Tip: Keep each guide to a single page. If a topic needs more depth, consider splitting it into multiple focused guides.

Additional Details

You can include lists, callouts, images, and any standard HTML content in your guides. The styling above gives a clean, readable baseline.

  • Use <h2> tags for section headings
  • Use the .callout class for important notes
  • Keep content concise and actionable
